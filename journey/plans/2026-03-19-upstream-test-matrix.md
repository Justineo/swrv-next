# Upstream Test Matrix

Date: 2026-03-19
Status: Active

## Purpose

Track every upstream `swr` and legacy `swrv` test file against `swrv-next`, so the remaining work is explicit and auditable.

Status legend:

- `Covered`: current `swrv-next` tests already exercise the applicable behavior closely enough.
- `Partial`: some important behavior is covered, but the upstream file still has applicable cases we have not ported or adapted yet.
- `Deferred`: intentionally out of the active lane for now.
- `Not applicable`: React-only or internal-only in a way that should not be ported to Vue.
